# Definition of Done

A feature is only done when:

- canonical ownership is clear
- backend contract exists and is wired
- UI works on the intended route
- empty/loading/error states exist
- no mock production data remains
- typecheck passes
- build passes
- critical smoke test passes
- follow-up gaps are explicitly listed

## Not enough
These alone do not mean done:
- it compiles
- it renders
- it looks close
- the happy path works once
