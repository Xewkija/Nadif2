# Project Vision

## What Nadif is
Nadif is a premium, multi-tenant operating system for cleaning businesses.

It must serve three primary personas in parallel:
- staff
- customers
- providers

## Product principles
- premium UX over generic admin UI
- one canonical owner per domain
- multi-tenant safety from day one
- backend truth over client guesswork
- operational clarity over feature clutter

## What we are rebuilding for
- cleaner architecture
- explicit ownership
- fewer duplicate flows
- fewer regressions
- easier AI-assisted development
- stronger customer/provider parity with staff tools

## Non-goals
- shipping disconnected UI shells
- rebuilding everything at once with no sequence
- preserving legacy complexity for its own sake
